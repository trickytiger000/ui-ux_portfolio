Zoo Data Aggregator

A playful webpage created using React JS where users can sort, filter, and add animals from a fictional zoo to their "best friends" list!

Product:

"Zigzag Zoo"
Interactive Website

This website was created using React.js as part of a UI/UX class at Brown Univerity in Fall of 2022.

The project is meant to practice creating and deploying an interactive application that reads and displays data and changes appearence/content based on user input.
